Em manutenção
Operadoras API

Projeto onde foi construída uma API que contém os dados de contato das operadoras de internet dos clientes, pois na empresa havia uma necessidade de ter os dados centralizados para contato. A API funciona como uma agenda de contatos, também sendo feito um Front-end para Acessar os dados da API.

Esta API foi feita utilizando .Net, EF Core e inicialmente SQLite que foi migrado para MySQL. Podendo ser consumida com Get, Post, Put e Delete, pois desta forma os novos clientes poderiam estar registrando suas operadoras ou escolhendo elas dentro da lista das operadoras já registradas em nosso banco de dados.

Tecnologias utilizadas no projeto: HTML5, CSS3, Angular, SqlServer, EF Core e .Net Core
